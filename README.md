# Brain Punk

| 工具 | 版本 | 备注 |
| :--- | :--- | :--- |
| Unity3D | 6000.0.59f2 | 开发引擎 |
| GF | 1.0.0 | 游戏开发框架|
| GuruSDK | 3.1.0 | GuruSDK 业务框架|

## 初始化项目

可直接执行命令

```shell
install_sdk.command
```

项目会进行：

- GuruSDK 安装
- GF 初始化
