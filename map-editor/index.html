地图编辑器

新建元素画布

输入正方形边长 N，在格子上点选该元素占用的格子（可多格，如 2×2、L 形等）。右键一个已选格可设为该元素旋转锚点。

不选图片时：用所选颜色填充格子并描边，名称会叠在占用区域的正中央（与地图中一致）。

实时预览

与地图中放置后的效果一致；改格子、名称、颜色或图片后会立即更新。

请先在上方格子里至少选择一格

元素列表

保存后会写入本机浏览器，刷新页面仍保留；亦可导出 JSON 备份。

    场景

    先选元素再放置，或选取消选择后拖拽或点击空地移动实例。R 旋转当前选中（放置预览或已选实例）。右键删除。地图列表在右侧。

    地图列表

    保存在本机浏览器；顶部可导出工作台备份。切换地图前若有未保存更改将提示。

      可放置元素

      横向点选一项，再到地图上点击格子放置（点击格即元素锚点格）。也可拖拽到地图上。

        编辑元素

        点右上角 × 或遮罩关闭将放弃修改。右键已选格可设置旋转锚点；保存后会更新元素库并同步地图列表中的内嵌定义。
